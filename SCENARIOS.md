## 🔧 设备标注配置方法

在 `bridge/config.py` 的 `devices` 节点添加设备映射:

```python
devices = {
    "7254adb5": {
        "name": "客厅监控",
        "location": "客厅",
        "role": "摄像头+传感器+音箱",
        "wifi_ip": "192.168.40.84"
    }
}
```

Dashboard 和日志系统会自动使用友好名称显示。
